# Голосовой бот для опросов

Реализовано с помощью библиотеки SpeechRecognition

Для тестирования нужно подключить микрофон

### Как запустить

Установить зависимости из файла requirements.txt:
```
pip install -r requirements.txt
```

Запустить бота:
```
python main.py
```
